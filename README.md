Film Industry Data Analysis Project
•Conducted exploratory data analysis (EDA) on a movies dataset to extract insights about revenue, budget,
ratings, and genres.
• Cleaned and transformed raw data using Python (Pandas, NumPy) and Excel to handle missing values and
inconsistencies.
•Analyzed key performance indicators such as profitability, genre trends, and top-performing movies by
revenue and rating.
• Built visualizations using Matplotlib / Seaborn to identify trends and correlations (e.g., budget vs.revenue).
•Generated actionable insights to support data-driven decision-making in the film industry.
